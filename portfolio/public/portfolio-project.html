Arquitectura

El portfolio es parte de una infraestructura más amplia gestionada completamente con Terraform como Infraestructura como Código (IaC). La arquitectura se despliega en AWS, utilizando una instancia EC2 de Ubuntu como servidor base, donde se ejecutan múltiples microservicios en contenedores Docker. El tráfico se gestiona mediante Traefik como reverse proxy, que maneja las reglas de enrutamiento para cada servicio. La resolución DNS se realiza a través de Route 53, apuntando al endpoint público de la instancia EC2. Para el análisis de uso y métricas, se ha integrado Umami Analytics, permitiendo un seguimiento detallado de las interacciones de los usuarios. Toda la infraestructura, incluyendo redes, seguridad, y recursos de AWS, está definida y versionada en código Terraform, permitiendo un despliegue consistente y reproducible. Para más detalles sobre la implementación de la infraestructura con Terraform, puedes visitar el proyecto de Terraform.
